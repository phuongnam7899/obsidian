# Local storage
- Max 5MB
- No expired time
- No send data to server
# Session storage
- Auto clear when close tab
- No send data to server
# Cookie
- When open app, browser auto send cookie to server to know what user did last time
- Max 4kB
- Mainly read in server
- Timeout: configable
